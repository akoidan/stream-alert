- Install OBS
- Install full ffmpeg build
- Specify pushcut token and notification name in config/default.json
- Specify camera name that OBS creates as virtualcamera
- Capture game window from OBS and click Start Virtual Camera
- Run `yarn install`
- Run the program with `yarn start`
- Modify `diffThreshold` - amount of pixes required to change in the video
- Modify `treshold` (0..1) - how significant change should be. Lower values require bigger changes per pixel
